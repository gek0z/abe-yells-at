# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Email:** security@riccardo.lol

Please include:

- A description of the vulnerability
- Steps to reproduce it
- Any potential impact

I will respond within 48 hours and work with you to address the issue before any public disclosure.

## Scope

This policy applies to:

- The `abe-yells-at` npm package
- The web app at [abeyells.at](https://abeyells.at)
- This GitHub repository

## Out of Scope

- Third-party dependencies (report directly to the maintainer)
- Social engineering attacks
